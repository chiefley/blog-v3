# WA1X Site Assets

Use this folder for WA1X-specific CSS, functions/hooks, and future React bundles.
- Upload bundles from `components/dist/wa1x/` to `/wp-content/uploads/react-components/wa1x/` when available.
- Add WA1X Additional CSS in `css/` to keep GeneratePress tweaks versioned.
- Register WA1X components in `config/components.json`; the plugin will pick them up automatically.
